# 2023_BootCamp_FEC
 스마트 팜 제어 웹페이지 제작 프로젝트   
- 구현기능
   - 로그인 세션
   - LED 제어
   - 센서 측정 값 모니터링 기능  

위 프로젝트는 2023 SW전문인재양성사업 기업주도 프로젝트의 일환으로 개발된 프로젝트입니다. 

# Tech Stack 
 - Back_End 
   - Spring Boot 3.0
   - MySql
    
 - Front_End
   - HTML 5 
   - CSS 3
   - Thymeleaf
 
 - Device 
   - Arduino ESP32 (ET Board)
   - C++
